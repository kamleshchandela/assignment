HTML Entities & Symbols Reference

Introduction:
HTML entities are used to display reserved characters and special symbols that cannot be typed directly on a keyboard.

Currency Symbols 

Symbol: ₹   Code: &#8377;   Description: Indian Rupee
Symbol: $   Code: &dollar;  Description: US Dollar
Symbol: €   Code: &euro;    Description: Euro
Symbol: £   Code: &pound;   Description: British Pound
Symbol: ¥   Code: &yen;     Description: Japanese Yen

Mathematical Symbols

Symbol: ±   Code: &plusmn;   Description: Plus-Minus
Symbol: ×   Code: &times;    Description: Multiplication
Symbol: ÷   Code: &divide;   Description: Division
Symbol: ≤   Code: &le;       Description: Less than or equal
Symbol: ≥   Code: &ge;       Description: Greater than or equal

Arrows 

Symbol: ←   Code: &larr;     Description: Left Arrow
Symbol: →   Code: &rarr;     Description: Right Arrow
Symbol: ↑   Code: &uarr;     Description: Up Arrow
Symbol: ↓   Code: &darr;     Description: Down Arrow
Symbol: ↔   Code: &harr;     Description: Left-Right Arrow

Emoticons & Shapes

Symbol: ☺   Code: &#9786;   Description: Smiley Face
Symbol: ☹   Code: &#9785;   Description: Sad Face
Symbol: ❤   Code: &#10084;  Description: Heart
Symbol: ❤   Code: &#9733;   Description: Star
Symbol: ♠   Code: &spades;  Description: Spade

Greek Letters

Symbol: α   Code: &alpha;   Description: Alpha
Symbol: β   Code: &beta;    Description: Beta
Symbol: γ   Code: &gamma;   Description: Gamma
Symbol: π   Code: &pi;      Description: Pi
Symbol: Ω   Code: &Omega;   Description: Capital Omega
